Detailed Description

Represents a command to be executed by the shell.

command_t should be created using make_command(). A command_t created using this method must be free'd using free_command().

Field Documentation

cmd_args_t command_t::args

args is the arguments for the command, it contains size information in addition to command arguments.

char* command_t::bin

bin is the first part of the text that was entered into the console, the shell will attempt to search the path to find an executable and if none is found, the bin will be interpreted as a literal path to a file.
